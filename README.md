# Amplify Tutorial

1. Amplify Settings

   ```shell
       amplify init
   ```

2. Cognito 연동을 위한 amplify auth

   ```shell
       amplify auth add
   ```

3. aws amplify 관련 패키지 설치 (node 환경 amplify 플러그인, 리액트 전용 UI)

   ```shell
       yarn add aws-amplify @aws-amplify/ui-react
   ```
